INLINE TAGS

This is a strong tag in HTML. This is a b tag in HTML. strong is used to specify text that is important. b is used to just make something bold

This is an i tag in HTML. It makes something visually italic. This is the em tag in HTML. The em tag is used to emphasize something. Like strong, it adds semantic importance for both visitors and web crawlers

This is a u tag in HTML. It underlines text. This is a small tag in HTML. It makes the text visually smaller. This is a sup tag: 53. It superscripts. This is a sub tag: 53. It subscripts.

All these tags are inline, meaning they exist on the same line, unlike block level tags that make their own line

Two visitors to a site: People, or web crawlers(ex: google). Strong and Emphasis should be used on things that you want google to know are important on your site. Use b and i tags when you just want the text to visually be bold or italic, without adding any importance
